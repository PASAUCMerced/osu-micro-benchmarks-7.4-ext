# osu-micro-benchmarks-7.4-ext

This is part of the [cMPI](https://github.com/skhynix/cMPI) project. It is a fork of the OMB [osu-micro-benchmarks-7.4](https://mvapich.cse.ohio-state.edu/benchmarks/), extended to test latency and bandwidth for one-sided MPI across multiple pairs.

Please refer to the [README](./README) for instructions on how to build and install the benchmarks.
